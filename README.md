## 📄 Descrição

Página web desenvolvida como projeto integrador do semestre, representando os projetos e conteúdos estudados ao longo do ano em uma **timeline responsiva e animada**, utilizando exclusivamente **HTML5 e CSS3** — sem JavaScript.

A página conta com uma seção de timeline com 9 projetos reais desenvolvidos durante o semestre, além de uma seção "Minha Evolução" detalhando a jornada de aprendizado.

---

## 🚀 Tecnologias utilizadas

- HTML5 semântico (`header`, `nav`, `main`, `section`, `article`, `footer`)
- CSS3 puro (sem frameworks)
  - Custom Properties (variáveis CSS)
  - Flexbox e Grid Layout
  - `@keyframes` e transições
  - Media Queries para responsividade
- Google Fonts: [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) + [Geist Mono](https://fonts.google.com/specimen/Geist+Mono)

---

## 🎬 Animações implementadas

| Nome | Descrição |
|---|---|
| `fadeUp` | Entrada suave de cards e itens da timeline |
| `slideRight` | Título do hero desliza da esquerda |
| `markerPop` | Marcadores surgem com efeito elástico |
| `lineDraw` | Linha vertical se desenha ao carregar a página |
| `pulse` | Marcador do projeto final pulsa continuamente |

---

## 📱 Responsividade

| Breakpoint | Layout |
|---|---|
| Desktop `> 1024px` | Timeline centralizada, grid de 3 colunas na seção de evolução |
| Tablet `768px – 1023px` | Grid de 2 colunas, card futuro ocupa linha inteira |
| Smartphone `≤ 767px` | Layout em coluna única, nav empilhada |
